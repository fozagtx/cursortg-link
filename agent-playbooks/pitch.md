# Cloud Agent Playbook — Pitch + Content

Build judge-facing assets: deck (`create-pitch-deck` / `pptx`), optional marketing video, social posts.
Do not invent traction metrics. Use only repo-verified facts.

## Synced skills this playbook expects

Before launching the Cloud Agent, run:

```bash
./scripts/sync-skills-to-repo.sh /path/to/target-repo --playbook pitch
```

Then commit `.cursor/` in the target repo.
