# Cheatsheet

## Pre-Trade Decision Rules

| Question | If yes | If no |
|---|---|---|
| Can I explain the buy in two sentences? | Continue checklist. | Do not buy. |
| Is the edge information, timing, narrative, community, or chart? | Size according to strength. | Treat as FOMO. |
| Would new buyers care about this information? | Edge may be tradable. | Information may be irrelevant. |
| Is the coin type clear? | Use matching exit logic. | Classify before entering. |
| Can I trade normally tomorrow if it goes to zero? | Size may be acceptable. | Size is too large. |
| Do I know what invalidates the trade? | Enter only if reward/risk fits. | No trade. |

## Coin-Type Logic

| Type | What drives value | Research priority | Holding mistake |
|---|---|---|---|
| Viral memecoin | Raw attention and narrative speed | Canonical identity, virality, supply risk | Holding after attention dies |
| Community coin | Persistent social coordination | Active members, content, raids, holder quality | Mistaking pump-only activity for community |
| Utility coin | Attention plus product/value mechanics | Team, product, revenue, tokenomics, narrative simplicity | Treating complex fundamentals as a pure meme |
| Ownership coin | Company valuation exposure concept | Company growth, terms, liquidity event logic | Selling like a memecoin too early |

## Bundling Red Flags

| Signal | Interpretation |
|---|---|
| One cluster controls large supply | Hidden single actor can dump |
| Top holder above ~3.5% in fresh trenches | Single sell can heavily damage chart |
| Many fresh wallets | Possible coordinated launch/control |
| Same funding origin/time | Wallets may share operator |
| Smooth up-only low-volume chart | Possible honeypot or controlled pump |
| Volume much lower than market cap on young coin | Supply may be tightly controlled |

## Market Risk Dial

| Conditions | Behavior |
|---|---|
| Majors strong, groups active, fresh coins reach high MC, outside attention rising | More aggressive, but still thesis-led |
| Mixed market, runners selective, narratives fragmented | Smaller size, higher standards |
| Majors down, volume dry, groups quiet, launches die early | Protect capital, trade less or not at all |
| After missed/fumbled big trade | Pause; avoid revenge entries |

## Chart Rules

| Setup | Action |
|---|---|
| Higher highs and higher lows | Uptrend; look for pullback entries |
| Lower highs and lower lows | Downtrend; avoid bottom-catching |
| Break above lower high | Possible bullish transition |
| Break below higher low | Bearish structure change |
| Clean impulse then pullback to 0.5-0.786 plus support | Potential entry zone |
| Fib drawn in sideways chop | Ignore the fib |
| Support breaks, retests, rejects | Setup likely invalid |

## Profit and Loss Rules

| Situation | Default |
|---|---|
| Up meaningfully and imagining bigger numbers | Sell some |
| Life-changing money on screen | Secure it now |
| Would not buy current size today | Reduce to justified size |
| Thesis broken | Exit or reduce |
| Loss creates urge to recover fast | Stop and review |
| Same mistake repeats | Create a concrete rule, not a vague intention |

## Security Rules

| Situation | Rule |
|---|---|
| New wallet | Write seed offline, make redundant backups |
| Significant holdings | Use cold wallet; consider multisig for very large amounts |
| Unknown dapp/link | Verify URL through official channels; use browsing wallet |
| Support DM asks for seed | Scam |
| Friend or KOL sends urgent link/CA | Verify through multiple channels |
| Mystery airdrop token | Do not interact until verified |
| Public portfolio brag | Security risk; avoid |
