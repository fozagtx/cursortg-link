# 48-Hour Hackathon Runbook (BPFG)

Use when you have a second day. Day 1 = SEE + spine. Day 2 = BUILD + proof.

## Day 1 — How to See (hours 0–24)

| Block | Hours | Action | Exit gate |
|---|---|---|---|
| Bootstrap | 0–1 | Village prompt + tools + roles | Project live |
| Commission | 1–2 | Voice → Brief Builder → full commission | Living Q solid |
| Map | 2–3 | Orchestrator mapper | 3 territories |
| Scout A | 3–6 | Full brief deep research + bank | Quotes exist |
| Scout B | 6–9 | Second territory + optional third model | Triangulation |
| Council | 9–11 | Full or lead-pair elders | Demo spine + cuts |
| Thin build | 11–18 | Spike core path only | Clickable aha |
| Story v1 | 18–20 | Storyteller pitch draft | Hook in field language |
| Sleep / reset | 20–24 | Bank everything; no new features | Archive saved |

## Day 2 — How to Build (hours 24–48)

| Block | Hours | Action | Exit gate |
|---|---|---|---|
| Re-see | 24–26 | Re-read bank; @Detective: is Q still right? | Confirmed or rewritten |
| Counter-study | 26–29 | Scout weakness of your own thesis | Risks listed |
| Build hard | 29–40 | Main product path | Demo reliable |
| Proof | 40–43 | Whitelist claims; multi-model check | 3 hard quotes |
| Polish | 43–46 | UX only on demo path | No new scope |
| Transmit | 46–48 | Rehearse ×3; submit | Shipped |

## Day-2 extras (only if Day 1 solid)
- [ ] Convert best elder insight into onboarding copy
- [ ] One custom scout for your niche community
- [ ] Counter-study: “what would falsify us?”
- [ ] Optional: second audience brief (do **not** merge into first)

## What not to do in 48h
- 11 custom scouts
- Full MCP/Commons wiring unless that’s the product
- Quant theater without sources
- Perfect multi-client village architecture
- Rebuilding the village OS instead of using the kit

## Demo narrative that matches the method (judges feel it)
1. Thin idea we started with  
2. Living question we found  
3. Field quotes / void language  
4. Elder cut (what we refused to build)  
5. Product moment that closes the void  
6. What remains CANDIDATE  

That story *is* sophistication — not more features.
