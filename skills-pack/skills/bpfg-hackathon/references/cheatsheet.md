# BPFG Hackathon Cheatsheet (1 page)

## Loop
`Voice → Living question → Full brief → Scouts (2+ territories) → Elders argue → Build void-closer → Pitch with field quotes → Archive`

## 5 layers (enter at 1)
Pre-verbal felt need → Encounter → Pre-conscious → Rationalization (surveys) → Behavior (clicks).  
**BPFG @ Layer 1.** Traditional/big data often arrive too late.

## Village stages
L1 Sense (bias) → L2 Synthesize/council (sycophancy) → L3 Shape/brief (drift) → T1 Strategy in your voice → T2 Portable skills → T3 Archive compounds.  
**MVP village = sensors + shapers (scouts + elders).**

## Living question test
Open in ≥2 directions? Primary audience clear? Not a feature list?

## Full brief must include
Living Q · territories · void language · shadow · filter-out · outputs · evidence rules (3 quotes, cite sources)

## Deploy rule
**Never summarize the brief before research.** Paste full commission into deep research.

## Minimum village
- 1 Project system prompt  
- Elders: Detective ⇄ Decider (+ Antenna optional)  
- Scouts: Reddit/forums + Reviews (or X + TikTok)  
- Archive outside chat  

## Elder calls
```
@Detective — question under the question
@Decider — cut scope for demo
@Antenna — fringe + void language for hero copy
@A and @B: converse, don't monologue
```

## Multi-model
Same full brief × 2–3 tools → through-line + productive disagreement.  
Route: Perplexity (Reddit/sources) · Grok (X) · Gemini (Scholar/YT) · Claude (village/skills)

## Insight → product
| Field signal | Ship |
|---|---|
| Void phrase | Hero copy |
| Workaround | MVP automation |
| Misclassification | Positioning |
| Who holds pain | Persona + channel |
| Identity refusal | Trust/aesthetics |

## Evidence
Keepers: intensity + multi-place echo + ≥3 quotes.  
Hallucinations = **candidates** → validate, don’t trash or ship blind.

## Anti-patterns
Generator-only AI · single sycophant chat · volume=truth · two audiences/one brief · summarize-then-research · automate insight · pitch without field language

## 15-min rescue
```
We're lost. Reconstruct living question. Name ignored void.
Smallest demo that closes it. Delete list for next 30 min.
@Decider leads; @Detective blocks only if question is wrong.
```

## Configs
Oracle+Self = default speed · Guide when murky · Explore ≠ validate

## Mantra
**Medium not generator. Village not factory. Telescopes not stars. You are the mother function.**
