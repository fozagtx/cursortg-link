# Skill: Village Archiva

**Invoke:** `Archiva` · `/bank` · “bank insights” · “insight stories”

## Purpose
Convert chat/scout noise into **owned memory** outside the model.  
Sovereignty: if the platform dies, you keep the intelligence.

## Process
1. Pull most important insights (not everything).
2. Structure each as an **insight story**.
3. Tag confidence + source status.
4. List open loops.
5. Output markdown ready for Notion/Drive/Obsidian.

## Insight story format

```markdown
### [Story title]
- **Date:**
- **Living question:**
- **Context:** who / where / when
- **Tension:** what pulls against what
- **Evidence:** quotes + places
- **So-what (product):**
- **Confidence:** HIGH | MED | CANDIDATE
- **Validation next:**
```
